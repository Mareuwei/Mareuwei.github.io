重温《武林外传》上

发表于 2019-08-03 更新于 2019-08-18 分类于 趣谈 阅读次数：
本文字数： 2.1k
还记得那一年的《武林外传》吗？

2006年，《武林外传》热播。 那时我们刚上初中，校园到处回响着主角们的口头禅：

白展堂的那招“葵花点穴手”，眼疾手快一个箭步戳到同学身上， 心虚作祟还会紧跟着一句：“谁动谁小狗”。这也是继小学鼠来宝之后的一次最成熟的单押。

佟掌柜一句“额滴神呀”，成为我们学习陕西方言的奠基石，她一开口就好像要讲大道理，还会顺便甩你脸上五个肉夹馍。

郭芙蓉的“排山倒海”，这招要是在现实中想要造成杀伤力，完全靠着吨位取胜，在操场上喊起来也比葵花点穴手有气势得多。

吕秀才最能装了，满肚子墨水却只能当个账房（没有说账房不好的意思）。上学的时候，只要哪个同学开始说那句“子曾经曰过”，你就知道，他要开始装b了。

还有和焦太郎神似的李大嘴，天天让别人照顾他七舅老爷的燕小六，有情有义却有点自大固执的邢捕头.....以及片头的音乐“嘿，兄弟，我们好久不见你在哪里？”

只要这音乐一响起，13年前发生在同福客栈的一幕幕又重新回到眼前。

很多人都说，再看一遍《武林外传》，发现它不只是一个喜剧，更像是把人生的百态搬到我们面前，其中的五味杂陈，只有我们长大后才能看懂。

刚毕业时，每个人都曾像郭芙蓉一样，山盟海誓要成为豪杰，成就一番大业，可其实呢，不过是自己不知天高地厚。

她刚来同福客栈那会儿，觉得自己有一身本事却无处施展，帮倒忙就算了，还非要拉上别人跟她比武，试问有几个普通人能顶得住她那招排山倒海？

终于，大家对于她的跋扈实在看不下去，白展堂作为武功高强的“盗圣”，语重心长的和她说了这些话：

练武是用来比的吗？是在它真正需要的时候派上用场。强盗来了你可以迎头痛击，路见不平你可以拔刀相助，威力的源泉，都是来源>于你心中的这股正气，即使他没有武功，同样可以天下无敌，你明白吗？

江湖不是打打杀杀，有人的地方就叫做江湖。只有内心正直，才能成为一代侠客。

然而，这个浑身正气的白展堂，轻功天下第二，指力世间屈指可数，在同福客栈里却“怂“得很：

看见捕快吓尿裤子，看见追风差点昏过去，他的形象基本和伟岸的形象搭不上边，多数时候就是一个心理素质有点差的“怂货”。

但还是觉得老白是帅的，记得他说起自己轻功输给天下第一楚留香时，辩解称：

“比赛那天，我光脚而且顶风。”

你看，他连臭屁都那么有味儿。

不是有那么一句话吗，顺境靠小郭，逆境靠老白，绝境靠秀才。

其实同福客栈三条汉子里，最不怂的就是看起来最弱不禁风的吕秀才。

当然，不光表现在他敢追小郭——那个一只手就能把他捏死的江湖女侠。

也不只是因为他一个秀才敢去扛一点红和降龙十八掌

是因为在整个同福客栈里，只有秀才敢直面心狠手辣的盗神姬无命，并靠着嘴炮把他终结了。

自此，一战成名，所到之处必响起战歌：是谁杀了我！而我又杀了谁！

也是从这一集，吕秀才的形象才开始丰满起来，成为了每个观众内心值得认可的关中大侠。

在这之前，他最骄傲的事也只有：我三岁识千字，五岁背唐诗，七岁熟读四书五经，八岁精通诗词歌赋。

当他过分沉溺于这些光环时，一旁的大嘴和老白就会说：“25岁穷得连饭都吃不饱。”“连祖产都卖给掌柜的开客栈了”

其实每次看见他意难平的样子，还是羡慕的，毕竟他内心的骄傲还没有磨平，他知道自己不是个普通人，是天才。

毕竟，我们做个普通人，已经足够筋疲力尽。

《武林外传》里，爱情自然也是个“重头戏”。13年后，以成年人的角度重新回顾时，总能发现自己恋爱时的缩影。

李大嘴设想的和挚爱蕙兰的婚后生活：她耕田来她织布，她挑水来我浇园。

怪不得蕙兰不肯接受李大嘴的爱，这份爱太沉重了，这谁顶得住啊。

大嘴也就在旁人这耍耍嘴皮子，逞逞大男子主义。实实在在的碰到蕙兰了，他可就怂成了另外一个人。

他不像秀才，秀才碰到自己喜欢的人，还能通过反省，再自信自如的表达自己的爱意。

大嘴不一样，大嘴是晴天霹雳，是电闪雷鸣，是恰恰恰。大嘴除了做菜，其他时候浑身上下都是自卑的影子。

而秀才呢，虽然勇于表达自己的爱意，可还是逃不过渣男本质，一边对郭芙蓉深情款款，一边又心系祝无双。

当大嘴问他：“如果让你再选一次，你会选谁？”时，秀才首先是毫不犹豫地说：“当然是选小郭啦！”

但然后，他又认真地思考了一下：“既然是假设，如果我当初两个都娶了呢？”

渣男本质之一：看似深情却又摇摆不一

这样比起来，白展堂和佟掌柜之间才更像是成年人的爱情，哪怕爱到了心灵深处，但又不肯说出口。

相比展红绫，老白记得第一次见面湘玉穿什么，会在自己母亲面前维护她，展红绫来了他也没走。在他心中，展红绫是初恋般的存在，但真正能陪他走过八十回的还是佟湘玉。

毕竟月亮再好，也是挂在天边的好，捧在手里可未必。

有几集佟掌柜和白展堂因为误会分手后，佟掌柜每天都强颜欢笑假装坚强，直到最后才忍不住吐露心声。

直到他走的那一刻起，我的心，就像是被几千把刀子同时割着 割一刀，疼一下 好不容易结了疤，又是一刀 我每天都在都在祈祷，忘了他吧 忘了就好了，忘了就解脱了，忘了就不疼了 可还是忘不了 醒了满脑子都是他，好不容易睡着了，梦里头还是他

小时候看着这几集，恨不得快点演完，觉得好矫情。很久后的某一天突然看懂了，才明白生活不止有开心，还有真真切切的难过。
